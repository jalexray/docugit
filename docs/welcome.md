# Welcome to DocuGit

This is a sample document to help you get started. Try editing it right now -- everything you see here is **live markdown** rendered as rich text.

## Try the Toolbar

Use the formatting toolbar above to style your text:

- **Bold** some text
- *Italicize* a word
- ~~Strike through~~ something
- Add `inline code`

## Make a List

Here's a checklist of things to try:

1. Edit this sentence -- just click and type
2. Add a new heading using the **H1/H2/H3** buttons
3. Create a bullet list with the **List** button
4. Insert a code block with the **Code** button
5. Press **Cmd+S** (or Ctrl+S) to save your changes
